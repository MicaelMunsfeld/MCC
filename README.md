# 🚗 MCC: Protótipo de Software Web para Gerenciar Concessionárias de Veículos

[![Artigo Publicado](https://img.shields.io/badge/Artigo-Publicado-green.svg)](https://www.unidavi.edu.br/) 
[![Status](https://img.shields.io/badge/Status-TCC%20Aprovado-blue.svg)]()

Sistema desenvolvido como Trabalho de Conclusão de Curso (TCC) de Sistemas de informação e publicado na **Revista Caminhos (Unidavi)**. O objetivo do sistema é facilitar e modernizar a gestão operacional de concessionárias e lojas de veículos (garagens).

O projeto atua no controle de entrada e saída de veículos, cadastro de clientes e lojistas, além de realizar a automação da precificação de mercado via APIs externas.

---

## 🌟 Diferenciais do Projeto e Artigo

* **Caso de Uso Prático:** O sistema foi desenhado para resolver dores de negócios reais de lojas de veículos (estoque, precificação e auditoria).
* **Integração com API Externa (Tabela FIPE):** Automação e sincronização de dados de marca, modelo, ano e valores de mercado em tempo real.
* **Pesquisa Acadêmica Aplicada:** Artigo publicado envolvendo levantamento bibliográfico de mercado e validação de protótipo de software.

---

## 🏗️ Arquitetura e Engenharia de Software

O projeto foi construído utilizando os pilares clássicos da Engenharia de Software para garantir manutenibilidade e baixo acoplamento:

* **Arquitetura:** MVC (Model-View-Controller)
* **Paradigma:** Programação Orientada a Objetos (POO)
* **Segurança:** Tratamento de entradas de dados, higienização de requisições e segurança nas consultas de banco.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem Principal:** PHP / JavaScript (Vanilla)
* **Banco de Dados:** MySQL / PostgreSQL
* **Ambiente:** Apache
